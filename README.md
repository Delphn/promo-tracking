# Run the code
```
node index.js
```

# Init new node project
```
npm init
```

### Useful way to use puppeteer
- you can launch it in full browser mode
```
puppeteer.launch({headless: false})
```


### Useful commands for starting a Node project

- init the project
```
npm init
```

- add gitignore
```
npx gitignore node
```

- add LICENSE
```
npx license mit>LICENSE
```
